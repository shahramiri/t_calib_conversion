How to use the file:
1. Copy and replace RawFrame.Tif from the following path:

\Calibration\FPD_FrameCaptureData\Frame\RawFrame.Tif

2. Run CalibMenu V1.4 or later and check "Flat Panel Detector" and "Use Previosuly Captured Images" checkboxes on the Start tab

3. On the "Frame Calibration" tab click on "Detect Device". 

4. Adjust the Cyan coloe ellipse to the grid and hit "Analyze"

5. The expected target Aspect Ratio from the analysis should be 1.10802 (1024/924)
